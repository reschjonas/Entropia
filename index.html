Quick Installation

1. Prerequisites

Ensure you have Go 1.24+ installed and UDP network access.

# Check Go version
go version

# Should show Go 1.24 or higher

2. Build from Source

# Clone the repository
git clone https://github.com/reschjonas/entropia.git
cd entropia

# Build static binary
go build -trimpath -ldflags="-s -w" -o entropia .

# Verify installation
./entropia --version

3. Quick Start

# Terminal 1 - Create a room (listener)
./entropia create

# Terminal 2 - Join via auto-discovery
./entropia join <RoomID>

# Or join with manual IP:port
./entropia join <RoomID> <ip:port>

üîê Security Reminder

Always verify identity fingerprints through a trusted side channel (phone, video call, etc.) before trusting messages. Entropia displays fingerprints every 60 seconds for easy verification.

How It Works

Entropia uses a carefully designed handshake and message flow to ensure quantum-safe, authenticated communication:

1. Discovery

mDNS, broadcast, or global lookup finds peer addresses

2. Handshake

Kyber key exchange establishes shared secrets

3. Authentication

Dilithium signatures verify peer identities

4. Secure Chat

XChaCha20-Poly1305 encrypts all messages
